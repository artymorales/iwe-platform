# IWE Curriculum — программа тьюторинга

> **Цель:** Перестать воспринимать IWE как чёрный ящик. Понимать концепции, протоколы и механизмы.
> **Формат:** ~30 мин ежедневно в слоте саморазвития.
> **Метод:** Сократический диалог (вопрос → ответ → разбор → ссылки на источники).
> **Критерий усвоения:** агент оценивает понимание + пользователь подтверждает «понял, дальше».

## Правила тьютора

1. **Не лекция, а диалог.** Начинать с вопроса, ждать ответ, потом разбор.
2. **Ссылки обязательно.** После каждого разбора — где про это написано:
   - Оригинал FMT: `/tmp/FMT-exocortex-template/[путь]`
   - Наша адаптация: `~/iwe-platform/memory/protocols/[файл]`
3. **Примеры из реальной работы.** Не абстрактно — на задачах, которые мы уже делали.
4. **3 вопроса на проверку.** В конце каждой темы — мини-тест из 3 вопросов.
5. **Усвоено, когда:** агент оценивает ответы как уверенные + пользователь говорит «понял, дальше».
