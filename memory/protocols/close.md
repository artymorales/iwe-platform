# Протокол Закрытия (Close)

> Применяется в конце сессии, дня или недели.
> Цель: зафиксировать результат, сохранить знания, синхронизировать.

## Триггеры

- «Закрывай» / «закрывай сессию»
- «Закрывай день» → `~/iwe-platform/scripts/day-close.sh`
- «Закрывай неделю» → `~/iwe-platform/scripts/week-close.sh`
- Конец сессии (пользователь завершает разговор)

## Шаги (для сессии)

### 1. Что сделано
Краткое резюме: что было сделано, какой артефакт получился.

### 2. Capture
Проверить: появились ли новые знания, правила, различения?
Если да — записать в соответствующее место:
- `~/iwe-platform/memory/distinctions.md`
- `~/ds-knowledge-index/captures/`
- `~/ds-knowledge-index/drafts/`

### 3. Commit
Проверить `git status` в `~/iwe-platform/` (и других репозиториях,
которые трогали). Если есть незакоммиченные изменения:
- `git add -A && git commit -m "краткое описание"`
- `git push`

### 4. Отчёт
Краткий итог пользователю:
- Что сделано (1-2 предложения)
- Что захвачено (если было)
- Что дальше (рекомендация на следующую сессию)
